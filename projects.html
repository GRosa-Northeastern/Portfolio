Panel Calibration Project

This project was a three phase project, and I wanted to showcase some of the hand skills as well as detail in full some of the work I have done on this project. The main idea behind this project was to test the old calibration panel, wire the panel, and then test the new panel so we could compare the results. I was required to read an electrical drawing, solder wires together as well as to connectors, and crimp wires. Additionally, I was required to test post panels and organize the data using Python.

Other Projects

3D Printed Headphone Stand

I created a headphone stand for my Form and Structure course at Northeastern University. The task was to 3D print a practical object that could be used daily, which is why I chose a headphone stand that would fit many different brands of headphones. The model was created in SolidWorks and sliced in Cura.

First Year Engineering

The final project for my First Year Engineering course was to create a produc that teaches any STEM topic to elementary school students. This project was built around teaching the topic of conservation of energy, specifically potential energy and kinetic energy. This project incorporated laser cut wood and arduino components such as ultrasonic sensors and screen.
